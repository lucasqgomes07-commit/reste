🎮 Descrição do Projeto

Este projeto consiste em um sistema de jogos desenvolvido na linguagem C, executado no terminal. O programa apresenta um menu interativo que permite ao usuário escolher entre três jogos diferentes, com foco em lógica de programação e interação com o usuário.

Os jogos disponíveis são:

🧠 Pergunta e Resposta
🐍 Cobra na Caixa
⚔️ Gousmas War

O projeto utiliza conceitos importantes da linguagem C, como estruturas condicionais, laços de repetição, vetores, funções e geração de números aleatórios.

🕹️ Funcionalidades
🧠 Pergunta e Resposta
Quiz com 5 perguntas sobre a Primeira Guerra Mundial
Cada acerto vale 1 ponto
Exibe a pontuação final
Opção de jogar novamente
🐍 Cobra na Caixa
Jogo de dois jogadores
5 caixas com conteúdo aleatório:
Botão → vitória
Cobra → derrota
Jogadores alternam turnos até alguém ganhar ou perder
⚔️ Gousmas War
Jogo estratégico entre dois jogadores
Cada jogador possui duas “Gousmas”
Ações:
Atacar o adversário
Dividir valores entre suas gousmas
Gousmas com valor maior que 5 são eliminadas
Vence quem eliminar todas do adversário
